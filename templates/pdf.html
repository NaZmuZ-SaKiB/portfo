003 : Add watermark in PDF

Using this program you can add a custom watermark to a PDF file. You have to add the path of the main PDF and the PDF containing the watermark. It will automatically add watermark to all pages. I have used 'PyPDF2' module in the code.

"Watermark your PDF in the fastest way."

The main PDF

PDF containing watermark

Running the program in terminal

The output PDF with watermark

Here is the code


You have to specify the 'main pdf' and 'watermark pdf' file in the 'template' and 'watermark' section.


import PyPDF2 as pdf

#add your PDF files here:
template = pdf.PdfFileReader(open("path/filename.pdf", 'rb'))
watermark = pdf.PdfFileReader(open("path/watermark.pdf", 'rb'))
output = pdf.PdfFileWriter()

print('loading main PDF file......')
print('loading watermark......')

for i in range(template.getNumPages()):
page = template.getPage(i)
page.mergePage(watermark.getPage(0))
output.addPage(page)

print('adding watermark to the main PDF......')

with open('final.pdf', 'wb') as final_pdf:
output.write(final_pdf)

print('successfully added the watermark......')
print("check 'final.pdf' watermark......")
