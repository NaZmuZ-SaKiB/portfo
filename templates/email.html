001 : Gmail Sender

Send email using your gmail account. Nice simple program using python3.And I have also used 'email.message', 'smtplib' and 'get pass' module in the code.

"Send your email easy and quick."

Running the program and sending email

Recieved the email

Here is the code


You can run it in terminal


import smtplib
import getpass
from email.message import EmailMessage

my_email = EmailMessage()
user_name = input('Enter your name \t=')
email_address = input('Enter your email address\t=')
reciever_email = input('Enter reciever\'s email address\t=')
subject = input('Enter subject of email\t= ')
message = input('your message :\n\t')

password = getpass.getpass('Enter your email password \t= ')

print('Please wait.....\nStarting service.....')

my_email['from'] = user_name
my_email['to'] = reciever_email
my_email['subject'] = subject
my_email.set_content(message)

with smtplib.SMTP(host='smtp.gmail.com', port=587) as smtp:
smtp.ehlo()
smtp.starttls()
smtp.login(email_address, password)
print('Login successful.....')
smtp.send_message(my_email)
print('email sent.....')
print('quiting server.....')
smtp.quit()
